Guide to Jakarta Server Pages TCK 2.3 Documentation

The Jakarta Server Pages TCK 2.3 documentation includes the following:

  • The JSP 2.3 TCK Release Notes provides the information on the test harness, the test suite and the platforms tested.
    • If updates are necessary, new versions of these release notes will be made available throught the Jakarta EE Jakarta Server Pages Specification web site: https://jakarta-ee/specifications/server-pages/2.3/
  • The Jakarta Server Pages Technology Compatibility Kit, Version 2.3 User's Guide provides the information that you need to install, set up, and run the JSP TCK, Version 2.3. In addition, the guide provides the rules you must comply with to pass the JSP TCK.
  • The Javadoc Assertion List lists all the javadoc assertions that are tested by the JSP TCK.
  • The Specification Assertion List lists all the specification assertions that are tested by the JSP TCK.
