Guide to Jakarta Registries TCK 1.0 Documentation

The Jakarta Registries TCK 1.0 documentation includes the following:

  • The Jakarta Registries ReleaseNotes provide the information that you need to install, set up, and run the Jakarta Registries TCK, Version 1.0.

  • The Jakarta Registries TCK User's Guide also provides the information that you need to install, set up, and run the Jakarta Registries TCK, Version 1.0. In addition, the guide provides the rules you must comply with to pass the Jakarta Registries TCK.

  • The Javadoc Assertion List lists all the javadoc assertions that are tested by the Jakarta Registries TCK.

  • The Specification Assertion List lists all the specification assertions that are tested by the Jakarta Registries TCK.
