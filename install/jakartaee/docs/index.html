Guide to the Java EE 8 CTS Documentation

The Java EE 8 CTS documentation includes the following:

  • The Java EE 8 CTS Release Notes provides information about the test harness, the test suite, and the platforms tested.

    The Release Notes are available from Java Partner Engineering.

  • The Java Platform Enterprise Edition 8 Compatibility Test Suite User's Guide provides comprehensive user documentation, including instructions for installing and configuring the test suite, and using the JavaTest test harness to execute the tests. The User's Guide also contains the rules you need to pass for certification.

  • The Specification Assertion List lists all the specification assertions that are tested by the Java EE 8 CTS test suite.
